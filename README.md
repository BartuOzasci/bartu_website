# Bartu Website

Kişisel portfolyo ve projelerimi sergilediğim, modern teknolojilerle hazırlanmış web sitesi.

## Özellikler

- React tabanlı ana uygulama (src/)
- Statik HTML/CSS/JS ile hazırlanmış özel sayfalar (public/pages/)
- Modern ve responsive tasarım
- Kişisel bilgiler, projeler, iletişim formu ve CV bölümü

## Klasör Yapısı

- `src/` : React bileşenleri, yardımcı dosyalar ve ana uygulama
- `public/` : Statik dosyalar, görseller ve özel HTML sayfaları
- `public/pages/portfolio/portfolio.html` : CV ve detaylı portfolyo sayfası

## Kurulum ve Çalıştırma

1. Depoyu klonlayın:
   ```bash
   git clone <repo-link>
   ```
2. Ana dizine geçin:
   ```bash
   cd Bartu_Website
   ```
3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
4. Geliştirme sunucusunu başlatın:
   ```bash
   npm start
   ```

## Kullanılan Teknolojiler

- React
- Bootstrap 5
- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome

## İletişim

Her türlü soru ve iş birliği için iletişim bölümünü kullanabilirsiniz.
